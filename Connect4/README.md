# CONNECT FOUR  
![C](https://img.shields.io/badge/language-C-blue.svg)
![License](https://img.shields.io/badge/license-MIT-green.svg)

A connect four game enabling games such as Human vs Human / Computer vs Human / Computer vs Computer. Computers can have differents levels (random, medium or experts).

## Learning objectives

This project was developed to strengthen my understanding of:

- 2D arrays
- Modular programming
- Build automation using Make

## Features : 
- Human vs Human game
- Human vs Computer game
- Computer vs Computer game
- Modular C architecture
- Built with a Makefile

## Prerequisites :
- gcc

## Build :
```bash
make
```

## Usage :
```bash
./MonsterArena
```
